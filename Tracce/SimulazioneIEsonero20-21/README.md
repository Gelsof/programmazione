# Esercizio di preparazione al I Esonero di Programmazione (A - L) - A.A. 2020/2021
In questo repository viene riportata una possibile soluzione dell'esercizio di preparazione all'esonero presentato in aula il giorno 13/11/2020.
Il codice non è commentato. Come esercizio, si provi a fornire un commento per ogni blocco di codice riportato.
Si noti l'utilizzo dei costrutti studiati finora e si valuti se sono state implementate parti non esplicitamente riportate nella traccia.
Si ipotizzino ulteriori soluzioni al problema descritto in aula.
